Defination

Media Query CSS ka ek feature hai jo device ki screen size, width, height, orientation ya resolution ke hisaab se alag-alag styles apply karne ki permission deta hai. Iska use Responsive Web Design me kiya jata hai taaki website mobile, tablet aur desktop sabhi devices par sahi tarah se dikh sake. Media Query ki madad se hum different screen sizes ke liye alag CSS rules likh sakte hain. ✅

Advantage

Media Query website ko responsive banane me madad karti hai. Iski madad se web pages mobile, tablet aur desktop jaise alag-alag devices par sahi tarah se display hote hain. Yeh user experience ko behtar banati hai aur content ko screen size ke hisaab se adjust karti hai. Media Query website ki readability aur accessibility ko improve karti hai, jisse users ko har device par website use karne me aasani hoti hai
